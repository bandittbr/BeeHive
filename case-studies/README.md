# Case Studies

Casos de uso externos que demonstram aplicações construídas sobre o BeeHive.

Cada estudo de caso mostra:
- **Objetivo** do projeto
- **Workflow** BeeHive utilizado
- **Capabilities** envolvidas
- **Artifacts** gerados
- **Resultado** obtido

> Nota: Estes são exemplos de aplicações. O BeeHive como plataforma não conhece nenhum domínio específico.
