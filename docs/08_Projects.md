# 08 — Projetos

> **Natureza:** evolutiva. O Projeto é a **unidade central de valor** do BeeHive (P3). Tudo gira em torno dele.

---

## Definição
Um Projeto é um negócio ou iniciativa com identidade própria e resultados próprios. É o foco permanente da interface; os agentes existem para servi-lo.

## Composição de um Projeto
Um Projeto pode conter:
- **Identidade:** marca e identidade visual.
- **Presença:** contas de redes sociais e canais.
- **Conteúdo:** estratégia e produção.
- **Comercial:** produtos, marketplaces, afiliados.
- **Resultados:** métricas e finanças.
- **Força de trabalho:** os agentes designados ao Projeto.

> Nem todo Projeto usa tudo isso. Um Projeto pode ser tão simples quanto "criar um arquivo" (ver `10_Development.md`) ou tão complexo quanto um negócio digital inteiro (ver `05_Business.md`).

## Ciclo de vida (conceitual)
1. **Criação** — a partir de um objetivo definido pelo administrador (frequentemente via Conversa).
2. **Configuração** — identidade, escopo, agentes.
3. **Operação** — agentes executam tarefas; resultados se acumulam.
4. **Acompanhamento** — métricas alimentam decisões do administrador.
5. **Evolução / encerramento** — o Projeto cresce, pausa ou se encerra.

## Por que o Projeto é central
Colocar o Projeto no centro (e não os agentes) é o que mantém o sistema orientado a **objetivos e resultados** — a essência da visão (P2, P3).
