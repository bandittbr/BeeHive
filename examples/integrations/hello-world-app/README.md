# Hello World Application

Uma aplicação externa que usa o BeeHive **sem importar o Kernel diretamente**.

## Prova

Esta aplicação prova que:

1. Um terceiro consegue construir sobre BeeHive conhecendo apenas o SDK
2. Nenhuma modificação no Kernel é necessária para criar aplicações
3. A arquitetura está correta: Kernel + SDK + Plugins = Plataforma

## Como executar

```bash
npx tsx examples/integrations/hello-world-app/index.ts
```
